Commands



#start
To create giveaway quickly in the same channel
#end
To End Giveaway start
#create
Creates a giveaway in mentioned channel
#start-role
To create Giveaway-role and bot give the winners the role
#end-role
To end Giveaway-role
#edit
To change the prize/winners/time
#del
To Remove Giveaway
#reroll
To Choose a new winner
#pause
Pauses the giveaway
#unpause
Unpauses the giveaway
#countdown
Creates a Timer!
#endcountdown
To End Timer
#drop
Drop a prize for a time
#rank
To show your wins total and last prize
#dm
To enable or disable the winning DM
#set-lang
To change the bot language
#fix
To fix the stopped giveaway
#info-lang
To show what bot language is set to
#listblack
Show blacklisted users list
#setlog
Set Logs Channel
#emoji
Add emoji to your server
#blacklist
Blacklist a user from joining giveaways
#unblacklist
To remove user from blacklist
#setage
You can specify the age of the account that can participate in giveaway
#start-req Premium
To create giveaway with premium features
#level Premium
Shows specific user's level
#invites Premium
Shows specific user's invites
